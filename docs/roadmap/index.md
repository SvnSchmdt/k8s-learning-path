# Roadmap

This roadmap gives you a structured, phase-by-phase learning plan. It tells you what to learn, in what order, and how to verify that you have actually understood it.

---

## Structure

| File | Contents |
|------|----------|
| [Overview](00-overview.md) | Vision, learning philosophy, and phase overview table |
| [Learning Path](01-learning-path.md) | 8 detailed phases with goals, topics, and practical tasks |
| [Checkpoints](02-checkpoints.md) | Comprehensive self-assessment checklist for all 8 phases |

---

## Recommended order

1. Read the [Overview](00-overview.md) to understand the big picture
2. Follow the [Learning Path](01-learning-path.md) phase by phase
3. Use the [Checkpoints](02-checkpoints.md) to verify your progress before moving on

---

## The 8 phases

| Phase | Topic |
|-------|-------|
| 0 | Prerequisites & local setup |
| 1 | Container basics |
| 2 | Kubernetes fundamentals |
| 3 | Workloads: Pods, Deployments, ReplicaSets |
| 4 | Networking: Services, Ingress |
| 5 | Configuration: ConfigMaps, Secrets, Storage |
| 6 | Security, Helm, Kustomize, Observability |
| 7 | GitOps, Production Readiness, Next Steps |
