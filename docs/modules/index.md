## Module structure

Each module follows the same structure:

- **Ziel** — Learning goal
- **Warum** — Why this topic matters
- **Kernkonzepte** — Key concepts explained
- **Praxisaufgabe** — Hands-on tasks to run locally
- **Typische Fehler** — Common mistakes and how to fix them
- **Checkpoint** — Questions to verify your understanding
- **Definition of Done** — Clear completion criteria
- **Weiterführende Links** — Further reading from official sources

> **Note:** Module content is written in German (as the primary language of this learning path), with all technical terms in English. The website interface is in English.
